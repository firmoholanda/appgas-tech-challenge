## 🔨 instructions

to get started with the app, clone the repo:
```
$ git@github.com:firmoholanda/appgas-tech-challenge.git
```

navigate into the newly created folder:
```
$ cd appgas-tech-challenge
```

install dependencies:
```
$ bundle install
```

run rspec:
```
$ rspec spec
```

run aplication:
```
$ ruby ./app/app_gas.rb
```
